# student-assignment-detector
The Student Assignment Detector is a web-based application designed to streamline the academic submission process by automatically identifying and categorizing student assignments.
Built with a focus on automation in education, the project utilizes the following:
Core Functionality: It provides a structured system to manage and detect specific assignment parameters within a dedicated project_folder.
Tech Stack:
Python (58.7%): Used for the back-end logic, likely handling the detection algorithms or file processing.
HTML (41.3%): Used to create a clean, user-friendly interface for instructors or students to interact with the system.
This tool aims to reduce manual administrative work for educators, ensuring that submissions are correctly identified and organized for grading.
